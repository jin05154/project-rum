## Archive

### RUM의 주요기능
1. 중고나라, 번개장터, 당근마켓 통합검색 (전자기기 한정)
2. 허위매물 필터링: 매물의 신뢰도를 수치(ALC 지수)로 표시  
&nbsp;  
### RUM의 장점
- 이용자가 각각의 플랫폼에서 검색하는 번거로움을 덜어줌
- 허위매물, 광고성 글 없는 필터링된 매물검색 가능
- 중고거래 신뢰성 향상, 안전한 거래로 사기피해 줄임
